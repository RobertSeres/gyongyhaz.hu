## 2025-05-22 - Improved Mobile Menu Accessibility
**Learning:** Mobile menus often lack proper ARIA attributes and visual feedback for state changes. Using `aria-expanded`, `aria-controls`, and swapping icons provides better feedback for both screen reader and visual users.
**Action:** Implement a unified `toggleMenu` function to synchronize ARIA attributes and icon states.
