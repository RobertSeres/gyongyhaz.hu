# Palette's Journal - Gyöngy Ház

## 2025-05-14 - Initial Repository Review
**Learning:** The project is a static HTML site using Tailwind CSS and Font Awesome via CDNs. It lacks standard accessibility features like labels for form inputs and ARIA labels for icon-only links/buttons.
**Action:** Implement `sr-only` labels for inputs and `aria-label` for social links and close buttons to improve screen reader support without affecting the visual design.
