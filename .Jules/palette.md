# Palette's Journal - Gyöngy Ház

## Philosophy
- Users notice the little things
- Accessibility is not optional
- Every interaction should feel smooth
- Good UX is invisible - it just works

## 2025-05-15 - Initial Repository Scan
**Learning:** The site is a static single-page application for a beauty salon (Gyöngy Ház) built with Tailwind CSS, Font Awesome, and AOS. It features several interactive elements like a newsletter modal, review popup, and coupon modals, but many icon-only buttons and form inputs lack proper accessibility attributes (labels/ARIA).
**Action:** Implement micro-UX improvements focusing on accessibility (ARIA labels, form associations) and feedback (loading states for form submissions) to make the interface more inclusive and professional.
